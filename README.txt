Qué es

Visualizar las encuestas y mapas de MdP entre todos usando Inkskape + Jessyink para crear una presentación que funciona tanto offline como online
Cómo funciona

Inkscape es una herramienta de ilustración vectorial libre, que trabaja con el formato SVG.

Jessyink es un plugin que incorpora javascript para conertir cada capa en una slide, pudiendo crear una presentación interactiva.

Si bien cada grafico se puede ver directamente, uso una web para hacerlo más funcional, y apto para mobiles.
